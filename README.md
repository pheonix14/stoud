## Features
- **Simultaneous Streaming**: Stream to YouTube, Twitch, and Custom RTMPs at the exact same time using our efficient FFmpeg backend.
- **Headless 24/7**: Close your browser, shut down your PC. The Stoud Python backend handles the queue and streams independently.
- **Anti-Sleep Engine**: Built-in self-pinging systems to keep free cloud hostings (like Render) awake and running.
- **Live Overlays**: Add your own transparent PNGs as live watermarks or frames.
- **Open Source**: 100% free and open source.

## Installation

```bash
# Clone the repository
git clone https://github.com/pheonix14/stoud.git
cd stoud

# Install dependencies
npm install
pip install -r requirements.txt

# Build the frontend
npm run build

# Start the Python engine
python main.py
```
